# 仿知乎

## 技术栈

`golang` `go-zero` `mysql` `redis` `kafka` `es` `grpc`  `etcd`	`docker`

## 项目特点

- 采用微服务架构设计，从单体架构到微服务架构
- 高并发，模拟高并发场景，面对高并发时的性能优化
- 缓存异常的处理，穿透、击穿、雪崩、缓存数据库一致性保证
